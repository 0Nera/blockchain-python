# blockchain-python

Простейший пример блокчейна на python
 
```bash
python3 -m venv env
```

```bash
source env/bin/activate
```

```bash
pip3 install -r requirements.txt
```

Запуск сервера:

```bash
python3 server.py
```

Запуск кошелька:

```bash
python3 wallet.py
```

Запуск майнера:

```bash
python3 miner.py
```
